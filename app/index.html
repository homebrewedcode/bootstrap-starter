BOOTSTRAP STARTER

This is my basic starting point for Bootstrap 4 enabled sites. I use NPM for all dependencies and Gulp for a build tool.

Setup

First off, you will need NPM installed on your system. This set of steps should bet you up to speed:

  • Clone this repository onto your system
  • Run npm install
  • Now you should be ready to use this boilerplate code as the bootstrap styles are imported into custom_bootstrap.scss and .js dependencies have already been populated
  • Custom css should be written into app/scss/custom.scss
  • Custom javascript should be placed in app/js/app.js
  • Start building the homepage in app/index.html and place any new desired .html files in the app directory

Development

During development, the basic workflow should go as follows:

  • Run the default gulp command with gulp to start the the browserSync server
  • A new browser tab will open in your default browser
  • Whenever a file is saved, the browser will be refreshed automatically
  • If dependencies are updated, you can run gulp build:dev which will clean out and repopulate the bootstrap js dependencies from node_modules
  • Any new js dependencies will need to manually be added to the gulp tasks and added to the html files between the comments for css or js files

Final Build

Once you are ready to build your app for production, follow these steps:

  • Run gulp build to populate the dist folder with your files
  • All scss will be compiled and css and js files will be minified into one css and js file. Images and fonts will also be optimized and copied over to dist in their respective folders
